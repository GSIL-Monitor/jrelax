首先需要获取到树对象，然后才能调用方法 $('el').jstree(true).方法名()

参数中的节点表示：DOM元素、jQuery对象、jQuery表达式、节点ID值

名称 参数 描述
create 节点， 参数 创建树
destroy 销毁树
reference 节点 获取jstree对象， 等同于$("el").jstree(true)
get_container 获取jQuery容器
get_node 节点，DOM 获取节点数据
get_path 节点，分割字符，ids 获取当前节点路径
get_next_dom 节点 获取下一个节点
get_prev_dom 节点 获取上一个节点
get_parent 节点 获取父节点
get_children_dom 节点 获取子节点（一级）
is_parent 节点 是否有子节点
is_loaded 节点 节点是否已经加载，远程数据时生效
is_opened 节点 节点是否已经展开
is_closed 节点 节点是否已经关闭
is_leaf 节点 节点是否不包含子节点
load_node 节点，回调函数 加载节点。 触发 load_node.jstree
load_all 节点，回调函数 加载所有节点。触发 load_all.jstree
redraw 是否重绘所有节点 重绘节点
open_node 节点 展开节点。触发 open_node.jstree after_open.jstree before_open.jstree
close_node 节点 关闭节点。触发 close_node.jstree after_close.jstree
toggle_node 节点 展开/关闭节点
open_all 节点，动画显示 展开所有节点。触发 open_all.jstree
close_all 节点 关闭所有节点。触发 close_all.jstree
is_disabled 节点 是否禁用
enable_node 节点 启用节点。触发 enable_node.jstree
disable_node 节点 禁用节点。触发 disable_node.jstree
is_hidden 节点 是否隐藏
hide_node 节点 隐藏节点。触发 hide_node.jstree
show_node 节点 显示节点。触发 show_node.jstree
show_all 节点 显示所有节点。触发 show_all.jstree
hide_all 节点 显示所有节点。触发 hide_all.jstree
select_node 节点，是否触发changed.jstree事件，prevent_open 选中节点。触发 select_node.jstree changed.jstree
deselect_node 节点 取消选中节点。触发 deselect_node.jstree changed.jstree
select_all 节点 选中所有。触发 select_all.jstree changed.jstree
deselect_all 节点 取消选中所有。触发 deselect_all.jstree changed.jstree
is_selected 节点 是否选中
get_selected 是否返回完整节点数据 获取选中的节点，默认返回id集合
get_top_selected 是否返回完整节点数据 获取选中的第一级节点，默认返回id集合
get_bottom_selected 是否返回完整节点数据 获取选中的最后一级节点，默认返回id集合
refresh 跳过加载显， 忽略状态 刷新树。触发 refresh.jstree
refresh_node 节点 刷新节点。触发 refresh_node.jstree
set_id 节点，ID 设置ID值。触发 set_id.jstree
get_text 节点 获取节点的显示文本
get_json 节点 获取节点的JSON格式数据
create_node 上级节点，节点数据，位置（first/last），回调函数，是否已经加载 创建节点。触发 create_node.jstree
rename_node 节点，显示文本 重命名节点。触发 rename_node.jstree
delete_node 节点 删除节点。触发 delete_node.jstree
move_node 节点，上级节点，位置（first/last），回调函数，是否已经加载，跳过redraw 创建节点。触发 move_node.jstree
copy_node 节点，上级节点，位置（first/last），回调函数，是否已经加载，跳过redraw 复制节点。触发 copy_node.jstree
cut 节点 剪切节点。触发 cut.jstree
copy 节点 复制节点。触发 copy.jstree
get_buffer 获取粘贴板中的节点，来源自cut和copy
can_paste 是否可以粘贴
paste 节点 粘贴节点。触发 paste.jstree
clear_buffer 节点 清空粘贴板中的节点。触发 clear_buffer.jstree
edit 节点, 默认显示文本，回调函数 编辑节点
set_theme 皮肤名称，皮肤地址 设置皮肤。触发 set_theme.jstree
get_theme 获取皮肤，返回皮肤名称
show_stripes 显示奇偶行不同背景色。触发 show_stripes.jstree
hide_stripes 取消显示奇偶行不同背景色。触发 hide_stripes.jstree
toggle_stripes 切换奇偶行不同背景色
show_dots 显示点。触发 show_dots.jstree
hide_dots 取消显示点。触发 hide_dots.jstree
toggle_dots 切换显示点
show_icon 显示图标。触发 show_icon.jstree
hide_icon 隐藏图标。触发 hide_icon.jstree
toggle_icon 切换显示图标
show_ellipsis 显示省略号。触发 show_ellipsis.jstree
hide_ellipsis 隐藏省略号。触发 hide_ellipsis.jstree
toggle_ellipsis 切换显示省略号
Checkbox 复选框
show_checkboxes 显示复选框
hide_checkboxes 隐藏复选框
toggle_checkboxes 切换显示复选框
is_undetermined 节点 是否为半选中状态
disable_checkbox 节点 禁用复选框。触发 disable_checkbox.jstree
enable_checkbox 节点 启用复选框。触发 enable_checkbox.jstree
check_node 节点 选中复选框。触发 check_node.jstree
uncheck_node 节点 取消选中复选框。触发 uncheck_node.jstree
check_all 全选所有复选框。触发 check_all.jstree
uncheck_all 取消选中全部复选框。触发 uncheck_all.jstree
is_checked 节点 是否选中
get_checked 返回全部数据 获取选中的节点
get_top_checked 返回全部数据 获取选中的第一级节点

Contextmenu 右键菜单

search 搜索
search 关键字，是否本地搜索 搜索

state 状态

types
