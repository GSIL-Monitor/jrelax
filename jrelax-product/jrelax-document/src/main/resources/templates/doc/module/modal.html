参数

名称 类型 描述 默认值
id string 弹窗的唯一标识，不传入时自动生成
title string 标题
content string 静态内容，remote=false时有效
size string 窗口大小，可选项：modal-lg、modal-md、modal-sm modal-md
border string 边框样式
theme string 弹窗皮肤：可选项：
  • modal-primary
  • modal-danger
  • modal-info
  • modal-warning
showHeader bool 是否显示头部 false
showFooter bool 是否显示底部 false
showOk bool 是否显示确定按钮 false
showCancel bool 是否显示取消按钮 false
showClose bool 是否显示关闭按钮 false
remote bool 是否远程加载，设置为true是，remoteUrl必须设置 false
remoteUrl string 远程地址
remoteUrl string 远程地址
loadText string 加载提示语 正在加载...
iframe bool 是否使用iframe模式加载远程地址 false
iframeHeight number iframe高度 300
cache bool 是否缓存，如果设置为true，则内容加载完成之后，第二次打开时将不再从远程获取。需手动设置id false
okText string 确定按钮文本 确 定
cancelText string 取消按钮文本 取 消
headerClass string 头部样式 no-b
headerAlign string 头部文本对齐，可选项：
  • text-left 左对齐
  • text-center 居中
  • text-right 右对齐
bodyClass string 内容区样式
footerClass string 底部样式 p10 no-b
footerAlign string 底部文本对齐，同头部文本对其
backdrop bool 是否显示遮罩层 true
keyboard bool 是否支持Esc按钮关闭弹窗 true
fullScreen bool 是否全屏显示 false
params object 用于窗口参数传递 false

事件

名称 参数 描述
onShow 显示前触发
onShown 显示后触发
onHidden 关闭后触发
onClose 关闭前触发
onLoaded 加载完成触发，remote=false时不触发
