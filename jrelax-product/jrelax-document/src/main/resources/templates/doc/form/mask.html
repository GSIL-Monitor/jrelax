输入格式化

对文本框的输入进行格式化，需引入 /framework/js/form/form-mask.js

使用示例

参数

使用data-mask、data-mask-alias、data-mask-regex设置参数

名称 类型 描述 默认值
mask string 采用通配符的方式对输入的格式进行限定，通配符包含：
  • 9 数字
  • a 字母
  • * 任意字符

如：年份格式为 yyyy年MM月dd日，对应的mask应为：9999年99月99日

使用{次数}或者{start,end}来进行动态数量控制，如 9{1,4} 表示1~4个数字 使用[9]来表示可选字符，如 999[9] 表示可以输入 三个数字或者4个数字

alias string 对常用的mask进行封装，可以快捷使用：包含：
  • currency 金额
  • decimal 浮点数
  • integer 整数
regex string 正则表达式
