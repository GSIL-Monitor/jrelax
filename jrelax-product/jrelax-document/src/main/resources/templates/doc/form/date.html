日期选择(格式化)
日期展示

参数

参数的配置方式为：在元素节点上设置 data-date-参数名="参数值"; 参数名为驼峰式时转换为下划线，如：startDate属性，配置则为：data-date-start-date

名称 类型 描述 默认值
autoclose bool 是否自动关闭 true
assumeNearbyYear bool 是否将两位的年份自动解析为四位年份，如 15-1-1解析为 2015-1-1 false
calendarWeeks bool 是否隐藏周 false
clearBtn bool 是否清楚已选择的日期的按钮。 如果设置autoclose为true，此参数不生效 false
container string 日期选择框的容器 "body"
datesDisabled string,array []
daysOfWeekDisabled string,array []
daysOfWeekHighlighted string,array []
defaultViewDate strin 默认显示的视图，分为三种：
  • year:年视图，并选中当前年
  • month:月视图，并选中当前月
  • day:日视图，选中当前日
 today
disableTouchKeyboard bool 触摸屏中，是否禁止弹出键盘 false
enableOnReadonly bool 日期选择的文本框设置为只读 true
endDate date，string 截止日期，此日期后的日期将被禁用，不可选择。 日期的格式必须和format属性相符
forceParse bool 是否强制解析文本框中的日期值，如果解析失败文本框的值会被改变 true
format string 日期格式，可用的配置项为：d, dd, D, DD, m, mm, M, MM, yy, yyyy
  • d, dd: 天数是否自动补0 如：5, 05
  • D, DD: 天数全称和简称 如：五, 五号
  • m, mm: 月份是否自动补0 如：5, 05
  • M, MM: 月份全称和简称 如：五, 五月
  • yy, yyyy: 年份2位和4位 如：15, 2015
 yyyy-mm-dd
immediateUpdates bool 选择年份或月份后，文本框的值是否立即变化 false
inputs array 用于实现日期范围选择 false
keepEmptyValues bool 仅在进行范围选择时生效，选择一个值后保持值不带到其他文本框中 false
keyboardNavigation bool 是否使用键盘的方向键进行调整 true
language string 语言,设置为其他语言之前需要导入语言包 zh_CN
maxViewMode string，number 设置最大可以显示的视图模式，可用配置项包括：
  • 0, days 天视图
  • 1, months 月视图、年视图
  • 2, years 年视图、十年视图
  • 3, decades 十年视图、世纪视图
  • 4, centuries 世纪视图、千年视图
 4，"centuries"
minViewMode string，number 设置最大可以显示的视图模式，可用配置项包括：
  • 0, days 天视图
  • 1, months 月视图、年视图
  • 2, years 年视图、十年视图
  • 3, decades 十年视图、世纪视图
  • 4, centuries 世纪视图、千年视图
 0，"days"
multidate bool 是否可以多选日期，多个日期用逗号隔开 false
multidateSeparator string 多个日期间隔符 ","
orientation string 显示方向，可用配置项包括：
  • auto 自动
  • top 上
  • right 右
  • bottom 下
  • left 左
 "auto"
showOnFocus bool 文本框获取焦点时立即显示日期选择框 true
startDate string，date 起始时间，起始时间之前的时间将不可用 当前时间
startView string，number 起始视图，可用配置项同上 0, "days"
templates object 模板
showWeekDays bool 显示周 true
title string 标题，显示在顶部
todayBtn bool 是否显示“今天”按钮 false
todayHighlight bool 当前日期保持高亮 false
toggleActive bool false
weekStart number 一周的开始， 0~6 （周日 ~ 周六） 1
zIndexOffset number z-index值，用于控制显示，防止被覆盖 10

方法

名称 参数 描述
destroy 移除datepicker
show 控制显示
hide 隐藏
update date 
                                        $('.datepicker').datepicker('update');
                                        $('.datepicker').datepicker('update', '2011-03-05');
                                        $('.datepicker').datepicker('update', '2011-03-05', '2011-03-07');
                                        $('.datepicker').datepicker('update', new Date(2011, 2, 5));
                                        $('.datepicker').datepicker('update', [new Date(2011, 2, 5), new Date(2011, 2, 7)]);
更新
setDate date 设置日期

事件

名称 参数 描述
show 显示后触发
hide 隐藏后触发
clearDate 日期被清除后触发
changeDate 日期更改后触发
changeMonth 从年视图切换为月视图时触发
changeYear 从十年视图切换为年视图时触发
changeDecade 从世纪视图切换为十年视图时触发
changeCentury 从千年视图切换为世纪视图时触发
